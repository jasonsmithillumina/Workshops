BaseSpace Tutorial : RNAExpress App: Hands-On Exercise

This tutorial will provide a walk-through the RNAExpress App in BaseSpace

 

Objectives:

 

  • Review Inputs for RNAExpress BaseSpace App
  • Inspect results from RNAExpress BaseSpace App

    •  

     

    Conventions used:
    Brackets <html element>
    Bold Text BaseSpace special terms
    Quotes“Literal On-Screen Text”
    U/I/BUser Action

    Step 1: Access BaseSpace  

    Open a Web Browser to http://goo.gl/qDHM7o

     

    ***If Needed:

  • Create BaseSpace account or Login
  • Accept Project
    (This contains completed RNA Express Analyses and Data)

    •  

    Step 2: Launch Inputs for RNA Express App

     

  • Click on “Launch App” RocketShip button

    •  

  • Select on “RNA Express App”

    •  

    Step 3: Specify Control Samples for RNA Express

     

  • Scroll to “Control Group” section

    •  

  • Click to “Select Samples(s)” <drop-down> menu

    •  

    Step 4: Select Samples for Analysis / Utilize Filter

     

  • Click on “Project” <drop-down> menu to filter displayed samples by Project name

    •  

  • Select Both “UHR” samples as input for Control Samples

    •  

  • Click on “Confirm” button

    		•

    Step 5: Specify Comparison (condition) Samples for Analysis

     

  • Select Both “Brain” samples as input for Comparison Samples

    •  

    Step 6: Navigate to completed RNA Express Analysis

     

    You will now be viewing a pre-computed RNAExpress Analysis from the project you accepted in Step 1

     

  • Navigate to “..RNA-Seq” Project withing BaseSpace

    •  

    Step 7: Open RNAExpress Analysis

     

  • Click RNA Express analysis : “RNA Express Brain v UHRR”
