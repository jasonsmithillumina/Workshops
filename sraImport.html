BaseSpace Tutorial : SRA Import Functionality

This tutorial will provide a walk-through of how to import data directly from the SRA (Sequence Read Archive) into a personal BaseSpace project

 

Objectives:

 

  • Create a BaseSpace Project
  • Launch a BaseSpace App/Analysis (SRAImport)
  • Review results from SRAImport App execution within a BaseSpace Project

    •  

     

    Conventions used:
    Brackets <html element>
    Bold Text BaseSpace special terms
    Quotes“Literal On-Screen Text”
    U/I/BUser Action

    Step 1: Access BaseSpace  

    Open a Web Browser to http://goo.gl/pZwo02

     

    ***If Needed:

  • Create BaseSpace account or Login
  • Accept Project if asked (We'll come back to this)
  • Click “Projects” button in Gray Navigation Bar at top of page

    •  

    Step 2: Create A Personal BaseSpace Project

     

  • Click on “New Project” Button

    •  

  • Enter Project Name “SRA Demo” when prompted

    •  

    Step 3: Open Your Personal BaseSpace Project

     

  • Click on your personal “SRA Demo” Project

    •  

    Step 4: Launch the SRA Import Analysis App

     

  • Click on “Launch App” RocketShip button

    •  

    Step 5: Select SRA Import App

     

  • Select “SRA Import” from the <drop-down menu>

    •  

    Step 6: Define Input for SRA Import App

     

  • Enter “DRA002295” into the <textbox>

    •  

    Notice the pre-population of the "SRA Demo" Project selection in the <drop-down menu> below the <textbox>.

     

    Any <Blue field> like this can be altered by the user for simple navigation using dynamic pre-poulated values

    Step 7: Launch the App/Analysis  

     

  • Click the <checkbox>

    •  

  • Push Continue <button> to launch the analysis

    •  

    ***Note:

  • The <checkbox> is required
  • The <white arrow> is the active part of the Continue <button> across most browsers

    •  

    Step 8: Verify App Launch 

     

  • Navigate back to the “SRA Demo” Project

    •  

  • Click on the “Analyses” tab under the “SRA Demo” Project

    •  

    Notice the pending execution flag.
    This will turn to when execution has commenced and once the analysis has finished.

     

    The analysis is executed in the cloud and the user has no control to advance this, unless using BaseSpace Onsite

     

    Step 9: Review Analysis Inputs Post-Launch 

     

  • Click on the “SRA Import ” Analysis

    •  

  • Click on the “Analysis” tab under the “SRA Demo” Project

     

  • Click on the “Inputs” tab under the “SRA Import ” Analysis

     

    Review provided <quo;DRA002295” input has been recorded.

    • Step 10: Review Results and Output Files 

  • Verify output files are present in the Samples tab of the “SRA Demo” Project after successful execution.

    •  

    These SRA sample files will be available as input for other BaseSpace apps.

     

    Congratulations. You are now officially a BaseSpace User!!
