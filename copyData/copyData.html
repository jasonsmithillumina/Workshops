BaseSpace Tutorial : Copy Data Between Projects

Back to Home Page

This tutorial will provide a walk-through of how to copy data from existing BaseSpace projects to “own” a local copy

 

Objectives:

 

  • Copy data from a BaseSpace Project
  • Review local data copy in personal BaseSpace Project

    •  

    Conventions used:
    Brackets <html element>
    Bold Text BaseSpace special terms
    Quotes“Literal On-Screen Text”
    U/I/BUser Action

    Step 1: Access BaseSpace  

    Open a Web Browser to http://goo.gl/pZwo02

     

    ***If Needed:

  • Create BaseSpace account or Login
  • Accept Project if asked (We'll come back to this)
  • Click “Projects” button in Gray Navigation Bar at top of page

    •  
    Create Project Tutorial (Opens in a new window)

    Step 2: Create A Personal BaseSpace Project

     

  • Use Project Name “Data Copy Demo” when prompted

    •  

  • Create Project Tutorial (Opens in a New Window)
    		•

    Step 3: Open Project with Existing Data

     

    Note: You MUST copy data FROM a project (source -> destination) “Pulling data” to a Project (destination <- source) is not permitted.

     

  • Click on your accepted project from Step 1 “*Metagenomics...”

    •  

    Step 4: Locate Samples to Copy Out

     

  • Click on “Samples” <sub menu> on the left

    •  

    Step 5: Select Samples to Copy

     

  • Select top <checkbox> to highlight all samples for copy.

    •  

    ***Note: “Copy to...” <button> NOW APPEARS

     

  • Click “Copy To...” <button>

    		•

    Step 6: Choose Destination Project for Copied Samples

  • Select “Copy Data Demo” Project as the destination

     

  • Click “Copy” <button>

    •  

     

    Step 7: Receive Notice of Successful Copy  

     

  • Watch for the green alert box in the upper right hand corner;

    •  

    Step 8: Verify Data Copy To Destination 

     

  • Navigate back to the “Copy Data Demo” Project

    •  

  • Click on the “Samples” <sub menu> on the left

    •  

  • Verify local copy of samples are present in “Copy Data Demo” project
